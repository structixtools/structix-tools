Two tools, two workflows

Same ecosystem, different strengths.

vAnthropic

Structix

A lightweight structural diff and duplicate detection CLI for TypeScript and C#. Great for developers who want quick entity-level insight instead of line noise.

  • Structural diff between git refs
  • Exact and structural clone detection
  • JSON, HTML, and AI prompt output
  • Optional Claude-powered explanations
Project README Detailed manual
vOpenAI

Analyzer

A workspace-based analysis tool built for engineering review, API evolution tracking, and PR-ready summaries.

  • Base/head repository comparison
  • API change reporting
  • Clone groups and clone drift
  • Markdown, JSON, PR-comment, and HTML output
Project README Detailed manual

Why this feels open-source

Local-first CLIs, plain reports, reproducible builds, zero SaaS lock-in.

Hackable

Rust codebases with clear module boundaries and tree-sitter parsing at the core.

Portable

Single-binary distribution model. Build locally or publish release binaries for each platform.

CI-Friendly

JSON, HTML, markdown, and PR-comment outputs make these tools easy to wire into reviews.
