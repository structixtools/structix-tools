Analyzer

A Rust workspace for structured comparison of C# and TypeScript repositories across git refs, designed for engineering review, API change awareness, and PR workflows.

Core capabilities

  • Base/head repository comparison
  • File changes, symbol changes, and API changes
  • Clone groups and clone drift detection
  • Outputs: Markdown, JSON, PR-comment, HTML
  • CLI + core workspace architecture

Build

cd vOpenAI
cargo build --release -p analyzer-cli

Binary output:

  • Linux / WSL: target/release/analyzer-cli
  • Windows: target/release/analyzer-cli.exe

Main command

analyzer-cli --repo /path/to/repo --base main --head HEAD --format markdown

Examples

cargo run -p analyzer-cli -- --repo . --base main --head HEAD --format markdown
cargo run -p analyzer-cli -- --repo . --base main --head HEAD --format pr-comment
cargo run -p analyzer-cli -- --repo . --base main --head HEAD --format html > report.html
