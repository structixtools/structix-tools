Structix

Structural code analysis CLI for TypeScript and C# codebases. It diffs code at the entity level across git commits and detects duplicate code, with optional AI narration via Claude.

Core capabilities

  • Structural diff between git refs
  • Entity-level change types: added, removed, modified, moved, renamed
  • Duplicate detection: exact clones and structural clones
  • Outputs: terminal, JSON, Markdown prompt, HTML
  • Optional Claude-powered explanations

Build

cd vAnthropic
cargo build --release

Binary output:

  • Linux / WSL: target/release/structix
  • Windows: target/release/structix.exe

Main commands

structix diff [REPO_PATH] [OPTIONS]
structix duplicates [REPO_PATH] [OPTIONS]

Examples

structix diff . --from HEAD~1 --to HEAD
structix diff . --from main --to HEAD --html -o report.html
structix duplicates .
ANTHROPIC_API_KEY=... structix diff . --from main --to HEAD --explain
