Structix Manual

Structix is a Rust CLI for structural code analysis of TypeScript and C# repositories.

What Structix does

Structix focuses on two workflows: structural diffing between two git refs, and duplicate detection in the current working tree.

Unlike line-based diffs, Structix compares entities such as functions, methods, classes, interfaces, constructors, enums, and properties.

Installation

cd vAnthropic
cargo build --release
  • Linux / WSL: target/release/structix
  • Windows: target/release/structix.exe
  • Requires git on PATH
  • ANTHROPIC_API_KEY only needed for --explain

Command overview

structix diff [REPO_PATH] [OPTIONS]
structix duplicates [REPO_PATH] [OPTIONS]

Structural diff

structix diff . --from HEAD~1 --to HEAD
structix diff /path/to/repo --from main --to feature-branch --html -o report.html

Useful flags:

  • --from <REF>
  • --to <REF>
  • --json
  • --prompt
  • --html
  • --explain
  • -o, --output <FILE>

Duplicate detection

structix duplicates .
structix duplicates /path/to/repo --prompt -o dedup-review.md

Duplicate categories:

  • Exact clones: same normalized source
  • Structural clones: same AST shape even if names differ

Typical workflows

structix diff . --from HEAD~1 --to HEAD
structix diff . --from main --to HEAD --html -o structix-report.html
ANTHROPIC_API_KEY=... structix diff . --from main --to HEAD --explain
structix duplicates .
